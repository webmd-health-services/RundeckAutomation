# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'RundeckAutomation'
Import-Module -Name 'RundeckAutomation'
```

To install privately:

```powershell
Save-Module -Name 'RundeckAutomation' -Path '.'
Import-Module -Name '.\RundeckAutomation'
```
